Python is a fun, yet powerful programming language that is now ubiquitous on almost all operating systems, with the exception of Windows. If you are unfortunate enough to be using Windows and need or want Python, then my installation guide may be of use. For any serious Python coders, I would recommend reading (and following) PEP 8.

Using Python

Prior to running python, make sure that python is on the path, if it is not follow my installation guide. There are two ways to run python: from a script and interactively.

Python Scripts

To run a python script, open a command prompt and type python script.py . On Windows, a python script can be ran by double-clicking its icon from Windows Explorer, but this is not recommended because the command prompt will quickly exit once execution is complete.

Interactive Python

To run python interactively, just type python from the command line. This is a fantastic way to learn a programming language, because it allows a rapid succession of trial and error exercises.
