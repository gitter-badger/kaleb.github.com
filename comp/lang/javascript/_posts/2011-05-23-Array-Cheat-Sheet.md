Note, all methods that are marked with an exclamation mark (!) will mutate their
object. All _italicized_ parameters are optional.

Static Methods
--------------

isArray(obj)

{% highlight js %}
[[],[1],2,"3"].map(Array.isArray);
    //-> [true,true,false,false]
{% endhighlight %}
